Drug Testing Using Machine Learning

By Nitya Tyagi — ML Developer

About the Project

This project focuses on developing a machine learning model to predict drug efficacy and potential toxicity based on molecular and biological data. The system utilizes supervised learning models and data preprocessing pipelines for reliable results.

Model Overview

  • Dataset: Drug response dataset (e.g., PubChem / Kaggle)
  • Model: Random Forest / XGBoost / Neural Network
  • Metrics: Accuracy, Precision, ROC-AUC
